# 6. Требования к системам удаленного банковского обслуживания

**Способ проверки**: Вручную
**Уровень**: 1С:Совместимо

## 6.1. Программный продукт для обмена электронными документами между системой "1С:Предприятие 8"

- **Способ проверки**: Вручную
- **Уровень**: 1С:Совместимо
- **1С:Совместимо**: да

6.1. Программный продукт для
обмена электронными документами между системой «1С:Предприятие 8» с одной
стороны и системой дистанционного банковского обслуживания или «шлюзом»
автоматизированной банковской системы с другой стороны должен соответствовать
правилам стандарта DirectBank, опубликованным на http://directbank.1c.ru/standart

## 6.2. Для программы допускается отсутствие "коробочного" вида продукта

- **Способ проверки**: Вручную
- **Уровень**: 1С:Совместимо
- **1С:Совместимо**: да

6.2. Для программы допускается отсутствие "коробочного" вида продукта.

## 6.3. Программа должна иметь документацию (допускается электронная версия), гарантийные обязательства по сопровождению (или бланк договора)

- **Способ проверки**: Вручную
- **Уровень**: 1С:Совместимо
- **1С:Совместимо**: да

6.3. Программа должна иметь документацию (допускается электронная версия), гарантийные обязательства по сопровождению (или бланк договора). В документации должно быть указано, как пользователь должен настроить программу для обмена данными с «1С:Предприятием», какой использует стандарт обмена и как производить обмен данными.

## 6.4. Для прохождения процедуры сертификации необходимо предоставить дистрибутив (при наличии) с демонстрационной базой

- **Способ проверки**: Вручную
- **Уровень**: 1С:Совместимо
- **1С:Совместимо**: да

6.4. Для прохождения процедуры сертификации необходимо предоставить дистрибутив (при наличии) с демонстрационной базой или обеспечить доступ к тестовому стенду для проверки взаимодействия.
