# 1C Standards Claude Skill

Скилл для [Claude Code](https://claude.ai/claude-code), который проверяет код 1С:Предприятие (BSL) на соответствие стандартам разработки.

## Что это

Набор правил из **Системы стандартов и методик разработки конфигураций для платформы 1С:Предприятие 8**, извлечённых из конфигурации **АПК (Автоматизированная проверка конфигураций) v1.2.9.124** от ООО "1С-Софт".

- 664 правила в 29 файлах
- Стандарты "Системы стандартов" (именование, структура модулей, запросы, производительность, безопасность и др.)
- Требования "1С:Совместимо" (1С:Предприятие 8.3)
- Требования по локализации, клиент-серверному взаимодействию, проектированию интерфейсов
- Встроенные проверки АПК (парсинг структуры модуля, документация параметров, права ролей, система исключений)
- Словарь соответствия русских и английских терминов 1С (1044 термина)

## Установка

Скопируйте папку `1c-standards` в `.claude/skills/` вашего проекта:

```bash
# Из корня вашего проекта 1С
mkdir -p .claude/skills
cp -r path/to/1c-standards .claude/skills/
```

Или клонируйте репозиторий и создайте симлинк:

```bash
git clone https://github.com/SolovievHub/1c-standards-claude-skill.git
cd your-1c-project
mkdir -p .claude/skills
ln -s /path/to/1c-standards-claude-skill/1c-standards .claude/skills/1c-standards
```

## Использование

После установки скилл автоматически активируется в Claude Code при работе с файлами `.bsl`. Можно также вызвать вручную:

```
> проверь код 1С на соответствие стандартам
> проверь файл src/cf/CommonModules/МойМодуль/Ext/Module.bsl
> проверь этот код по стандартам 1С
```

### Что проверяется

| Категория | Примеры правил |
|---|---|
| Оформление модулей | Структура модуля, области, комментарии к процедурам |
| Именование | CamelCase, осмысленные имена, булевы переменные |
| Работа с запросами | Запросы не в циклах, параметризация, оформление текста |
| Безопасность | Выполнить/Вычислить, хранение паролей, проверка прав |
| Производительность | Избыточные блокировки, оптимизация запросов |
| Клиент-сервер | Минимизация серверных вызовов, директивы компиляции |
| Локализация | Интерфейсные тексты, форматирование дат и чисел |
| Интерфейсы | Формы, командный интерфейс, элементы управления |

### Формат вывода

```
[ОШИБКА] Строка 42: Запрос выполняется внутри цикла
  Правило: Оптимизация запросов (std668)
  Рекомендация: Вынести запрос за пределы цикла, использовать временные таблицы

[ПРЕДУПРЕЖДЕНИЕ] Строка 15: Пустой блок Исключение
  Правило: Обработка исключительных ситуаций (std499)
  Рекомендация: Добавить запись в журнал регистрации

Итого: 1 ошибка, 1 предупреждение, 0 информационных
```

## Источник правил

Правила извлечены из справочника "Требования" конфигурации АПК v1.2.9.124 с помощью специальной обработки экспорта. Описание стандартов доступно на [its.1c.ru/db/v8std](https://its.1c.ru/db/v8std).

Если вы хотите сгенерировать правила самостоятельно из своей базы АПК (например, более новой версии), используйте: **[1c-standards-claude-skill-creator](https://github.com/SolovievHub/1c-standards-claude-skill-creator)**

## Лицензия

Стандарты разработки являются интеллектуальной собственностью ООО "1С-Софт". Данный проект предоставляет их в формате, удобном для автоматизированной проверки кода через Claude Code.
